# Architecture Decision Records (ADRs)

Registro leve das decisões de arquitetura com trade-off relevante, no formato
[Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).
Cada ADR é imutável: quando uma decisão muda, cria-se uma nova que **substitui**
a anterior (campo *Status*).
